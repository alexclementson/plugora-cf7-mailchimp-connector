Sync Contact Form 7 submissions straight into Mailchimp audiences — per-form mapping, tags, groups, double opt-in, GDPR consent and a polished Plugora admin UI.

== Description ==
Connect Contact Form 7 to Mailchimp in two clicks. Validate your API key, pick an audience per form, map fields with a friendly UI, and let Plugora handle the syncing — with full submission logs and helpful admin notices when something is misconfigured.

**Free**
* Mailchimp API key validation with live "Check key" button
* One audience per form
* Map name + email + standard merge fields
* Submission log (last 50 sends)
* GDPR consent checkbox support
* Double opt-in toggle

**Premium**
* Multiple audiences per form
* Tags, interest groups and segments
* Custom merge field mapping with conditional logic
* Update-existing-subscriber strategy
* Per-form retry queue with full log retention
* Zapier handoff for advanced routing

== Installation ==
1. Install &amp; activate Contact Form 7.
2. Upload the Plugora CF7 → Mailchimp ZIP via Plugins → Add New → Upload.
3. Activate.
4. Go to **Plugora → CF7 Mailchimp** to paste your Mailchimp API key.
5. Edit any CF7 form and use the new **MailChimp** tab to pick an audience.
